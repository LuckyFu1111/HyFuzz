## Statistical Analysis

### Confidence Intervals

All mean values are reported with 95% confidence intervals calculated using:
```
CI = mean ± (t_critical * SE)
where SE = stdev / sqrt(n)
```

### Coefficient of Variation (CV)

Measures stability/reproducibility:
- **CV < 5%:** Highly stable
- **CV 5-10%:** Acceptable variance
- **CV > 10%:** High variance (investigate)

Example:
- Medium mutation CV: 8.4% (acceptable)
- Indicates consistent performance across trials

### Effect Sizes

Cohen's d interpretation:
- **|d| < 0.2:** Negligible
- **|d| < 0.5:** Small
- **|d| < 0.8:** Medium
- **|d| < 1.2:** Large
- **|d| ≥ 1.2:** Very large

## Interpretation Guidelines

### When to Use Each Mutation Level

**Low Mutation:**
- Initial assessment of target stability
- Compliance testing (minimize false positives)
- Baseline establishment

**Medium Mutation:**
- General-purpose fuzzing (recommended)
- Balance between discovery and efficiency
- Production-ready testing

**Aggressive Mutation:**
- Maximum vulnerability discovery
- Security audits
- Research scenarios (willing to tolerate false positives)

### Duration Selection

- **30-60s:** Quick smoke tests, CI/CD integration
- **120-180s:** Standard fuzzing campaigns
- **300s+:** Comprehensive audits, research
